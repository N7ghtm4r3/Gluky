Report Creator

open class ReportCreator

The ReportCreator utility class is useful to create the reports pdfs with the given measurements. Some methods annotated as Returner in this context are intended like @Composable in Compose Multiplatform, so they represent a section of the pdfs and not properly a simple method and are defined in the javadocs as "Component"
